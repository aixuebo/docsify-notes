## 2.2 引用
+ 简单的引用 ```  ```
+ 特殊引用--可以根据格式,显示特殊的样式。
  + xml
  ```xml
   <?xml version="1.0" encoding="UTF-8"?>
  ```
  + java
  ```java
   import org.apache.flink.api.common.functions.FlatMapFunction;
  ```
  + 引用套用引用--注意 shell可以有回车样式
  ```java
   import org.apache.flink.api.common.functions.FlatMapFunction;
   ```shell
   输出:
    a b 
    c d
   ```
  ```
